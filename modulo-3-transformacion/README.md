# Módulo 3 — Transformación y Estadística Aplicada

**Semanas 5 y 6 · Clases 9 a 12**

## ¿Por qué este módulo?

Tener los datos no alcanza. Los datos reales están sucios: hay valores que faltan, registros duplicados, columnas con tipos incorrectos, y distribuciones que esconden anomalías. Este módulo es donde aprendés a limpiar, transformar y entender qué te están diciendo los números.

La estadística que vas a ver acá no es la del libro de matemáticas — es la estadística aplicada a decisiones reales.

- Limpiar un dataset sucio de forma sistemática
