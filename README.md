# 📊 Curso de Data Analysis con Python

> De los datos al insight. 12 semanas para convertirte en analista de datos.

## ¿De qué trata este curso?

Aprender a analizar datos no es aprender a programar — es aprender a hacerle las preguntas correctas a la información. En este curso vas a trabajar con datos reales, herramientas usadas en la industria, y vas a terminar con un proyecto propio que podés mostrar en tu portfolio.

**Duración:** 12 semanas · 2 clases por semana · 3 horas por clase  
**Total:** 24 clases

## Cómo está organizado cada clase

Cada clase tiene su propia carpeta con 2 archivos:

```

└── practica/

```
